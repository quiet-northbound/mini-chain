# Playground

Thư mục này dùng để thử nghiệm nhanh các ý tưởng nhỏ trước khi đưa vào `MiniChain.Core`.

## Lưu ý
- Code trong playground **không phải** production code
- Dùng để test concept, debug logic, hoặc sketch ý tưởng
- Khi ý tưởng đã rõ → chuyển vào `src/MiniChain/` với test đầy đủ
